# UI-KIT

## Installation

```
npm install @dv.net/ui-kit
```

```
yarn add @dv.net/ui-kit
```

## Usage

Example of importing into a Vue application:

```
<script setup lang="ts">
  import { UiButton, UiInput } from "@dv.net/ui-kit";
</script>
```

```
<style>
  @import "@dv.net/ui-kit/dist/style.css";
</style>
```
